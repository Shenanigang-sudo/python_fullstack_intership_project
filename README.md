# Flask User Management System

A simple web application built using **Flask** and **SQLite** that enables users to add and view user information through an intuitive interface. The project demonstrates the fundamentals of full-stack web development with Python, including database integration, server-side rendering, and form handling.
<img src="/images/Screenshot 2026-06-24 214212.png">
## 🚀 Features

* Add users with name and email
* Store data persistently using SQLite
* Display user records in a structured table
* Dynamic content rendering with Jinja2 templates
* Simple and clean user interface

## 🛠️ Tech Stack

* **Python**
* **Flask**
* **SQLite**
* **HTML**
* **CSS**
* **Jinja2**

## 📋 Database Schema

### Users Table

| Column | Type                              |
| ------ | --------------------------------- |
| id     | INTEGER PRIMARY KEY AUTOINCREMENT |
| name   | TEXT NOT NULL                     |
| email  | TEXT NOT NULL                     |
